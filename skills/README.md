# Skills

This directory contains repo-local Codex skills that capture reusable workflows for creator content production.

## Available Skills

- `talking-head-video-editing/`: transcript-guided talking-head video editing, subtitles, thumbnail generation, and platform packaging.

## How To Use

When working in Codex, point the agent to the skill path or ask it to use the talking-head video editing skill:

```text
Use skills/talking-head-video-editing/SKILL.md to edit this raw talking-head video.
```

The skill is intentionally stored in the repository so other creators can clone the project and reuse the same workflow without relying on a private local Codex setup.
